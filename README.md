# Mimi Mod for Super Battle Golf

## Video
[Gameplay video](https://imgur.com/a/LniKQXr)

## Info
Mimi Mod is a MelonLoader mod for Super Battle Golf.

## Main Features
- Assist toggle
- Auto aim camera while aiming
- Auto swing release while charging
- Live predicted trajectory
- Frozen predicted trajectory after shot
- Actual shot trail
- Impact preview window
- Coffee speed boost hotkey
- Nearest ball mode hotkey
- Unlock all cosmetics hotkey
- Config file for keys and visual settings

## Default Hotkeys
- `F` - toggle assist
- `F2` - coffee speed boost
- `F3` - nearest ball mode
- `F4` - unlock all cosmetics
- `RMB` - assist aim camera

## Visuals
- Actual trail
- Predicted trail
- Frozen predicted trail
- Impact preview enabled by default
- Default impact preview: `640x360 @ 60 FPS`

## Files

### Installer
For normal users.

- Includes a self-contained installer
- Installer already contains `MelonLoader`, `MimiMod.dll`, and `MimiMod.cfg`
- Just run the installer, select your game folder, and install

### Source
For users who want to inspect or build the mod.

- Includes full source code
- Includes `Source/MimiMod/MimiMod.csproj`
- Includes `compile_mod.bat`
- Includes required reference DLL files for building
- Project can be built from the source package and outputs `Mods/MimiMod.dll`

## Installation
1. Run `Installer/MimiModInstaller.exe`
2. Select your `Super Battle Golf` folder
3. Click `Install`

## Config
Config file: `Mods/MimiMod.cfg`

## Building From Source
- Open `Source/MimiMod/MimiMod.csproj`

or

- Run `compile_mod.bat`

Output: `Mods/MimiMod.dll`
